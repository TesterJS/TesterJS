# TesterJS

TesterJS is a simple library which let you compare in terms of performance one or several piecces of code. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

```
Node 6.11.0

```

### Installing

To start using TesterJS only need to follow the next steps

```
1- Download the project in local
2- Execute in the root folder    >_  npm start
```
